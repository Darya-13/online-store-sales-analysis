# online-store-sales-analysis
Data analysis project using Python, Pandas, Matplotlib and Seaborn. Includes data cleaning, exploratory analysis, visualizations and business insights based on online store sales data.
# Online Store Sales Analysis

## Project Overview

This project analyzes online store sales data using Python.

## Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Objectives

* Clean and preprocess data
* Analyze sales performance
* Explore monthly revenue trends
* Compare product categories
* Visualize key metrics
* Identify business insights

## Results

The analysis highlights sales dynamics, revenue distribution, customer behavior, and relationships between key variables through visualizations and statistical exploration.
